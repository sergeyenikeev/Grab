# Отзывы

## Цели
- Хранить:
  - мои отзывы/оценки (`review_type = my`)
  - публичные отзывы (`review_type = public`)
- Ограничение: не более 5 публичных отзывов на продукт.

## Текущее состояние MVP
- Есть таблица `reviews` и сервис `core/reviews` с лимитом.
- Внешние коллекторы отзывов по карточкам товара не включены в MVP (будут добавляться по источникам).

## Правила сбора
- Если программный сбор нестабилен или ограничен ToS:
  - режим `links_only` (сохраняется URL страницы отзывов)
  - либо ручной экспорт.
