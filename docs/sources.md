# Источники и ограничения

## Gmail
- Статус: поддерживается в MVP.
- Метод: Gmail API (`gmail.readonly`) через OAuth.
- Что извлекаем: тема, отправитель, текст/HTML, ссылки, вложения.

## Mail.ru / Yandex Mail
- Статус: поддерживается в MVP.
- Метод: IMAP over SSL + app-password.
- Что извлекаем: аналогично Gmail.

## Ozon / Wildberries / Яндекс Маркет / Мегамаркет / DNS / Ашан / AliExpress
- Статус в MVP: через письма.
- Direct API/экспорт: как этап расширения.
- Причина: соблюдение ToS и стабильности, без серого скрапинга.

## Прочие магазины
- Статус: через email-поток (подтверждения, чеки, уведомления).

## Публичные отзывы
- MVP: есть таблицы и сервисный лимитер (до 5).
- Сбор с карточек товара: добавляется по источникам отдельно, только где законно и стабильно.
- Fallback: `links_only` + ручной экспорт.
